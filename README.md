# Wild-Avocado-Project
Wild Avocado Project- Excel
The Data file contains data relating to weekly avocado sales in our American stores. You have been asked to present some data in Excel to see if there are any patterns or trends in the data. In particular, please answer the following questions: 

1.	Are there any macro trends over time relating to avocados across all regions?
2.	Which regions had the highest demand for avocados in 2018?
3.	Is there a difference in demand for conventional vs organic avocados?
4.	If the price of avocados is higher, how does this impact demand?

# Materials:
a .csv file outlining weekly avocado sales in American stores
